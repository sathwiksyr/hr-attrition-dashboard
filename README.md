# HR Employee Attrition Analytics Dashboard

## Project Overview
Employee attrition is a major challenge for organizations because losing experienced employees increases hiring costs, disrupts productivity, and reduces organizational knowledge. Companies need data-driven insights to understand why employees leave and how attrition varies across departments, roles, and salary ranges.

This project analyzes an HR dataset using Power BI to identify patterns in employee attrition and workforce distribution. The dashboard helps stakeholders visualize key factors influencing employee turnover and make informed decisions to improve retention strategies.

---

## Real-World Problem
Many organizations struggle with high employee turnover but lack clear visibility into the underlying causes. Without proper analytics tools, HR departments may not easily identify trends related to:

- Employee experience levels
- Department-wise attrition rates
- Salary distribution and its relation to turnover
- Workforce demographics

This project demonstrates how data analytics and visualization tools can transform raw HR data into actionable insights.

---

## Solution Approach
The solution uses Power BI to create an interactive analytics dashboard that processes employee data and presents insights visually.

Steps followed in the project:

1. Data Preparation
   - Cleaned and structured employee dataset
   - Removed unnecessary columns and handled missing values

2. Data Analysis
   - Calculated employee attrition rate
   - Analyzed distribution of employees by department, salary, and experience

3. Visualization
   - Created multiple interactive visualizations to explore workforce patterns

## Dataset Information
The dataset contains **1470 employee records** including attributes such as:

- Employee age
- Department
- Job role
- Salary
- Years of experience
- Attrition status

---

## Dashboard Features

- Employee attrition rate analysis
- Department-wise workforce distribution
- Salary distribution visualization
- Experience level analysis
- Interactive charts and filters

---

## Key Insights Found

- The overall employee attrition rate is approximately **16.12%**
- Some departments experience higher turnover compared to others
- Employees with lower salary ranges show relatively higher attrition rates
- Workforce distribution varies across job roles and experience levels

These insights demonstrate how analytics can support HR teams in developing better retention strategies and workforce planning.
